### Installation
**Fork** and **clone** this repository.

Then insall the dependencies

```
npm install
```

Then create the database

```
npm run db-init
```

Start webpack

```
npm run build-watch
```

To run the tests

```
npm test
```

To start the server

```
npm start
```

Then go to [http://localhost:1337/](http://localhost:1337/)
